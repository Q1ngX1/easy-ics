# 快速改色指南

## 🚀 5 分钟快速改色方案

### 步骤 1：确定新的颜色

选择一种主色和次色，例如：

- **绿色系**：主色 `#10b981`，次色 `#059669`
- **橙色系**：主色 `#f59e0b`，次色 `#d97706`

### 步骤 2：打开 VS Code 全局搜索替换

使用快捷键 `Ctrl + Shift + H` 打开"查找替换"面板

### 步骤 3：逐一替换

按照下面的顺序，在 `frontend/src/styles` 目录中进行替换：

## 📋 逐步操作指南

### 示例：从蓝紫改为绿色

1. **打开 VS Code**
2. **按 Ctrl + Shift + H** 打开替换面板
3. **第一次替换：**
   - 查找：`#667eea`
   - 替换为：`#10b981`
   - 点击"全部替换" ✓
4. **第二次替换：**

   - 查找：`#764ba2`
   - 替换为：`#059669`
   - 点击"全部替换" ✓

5. **第三次替换：**

   - 查找：`102, 126, 234`
   - 替换为：`16, 185, 129`
   - 点击"全部替换" ✓

6. **第四次替换：**

   - 查找：`118, 75, 162`
   - 替换为：`5, 150, 105`
   - 点击"全部替换" ✓

7. **保存所有文件** Ctrl + Shift + S
8. **刷新浏览器** F5 查看效果

## ⚡ 进阶：创建配色变量文件

如果要支持多主题，可以创建 `src/styles/themes.css`：

```css
/* 默认主题（蓝紫） */
:root {
  --primary-color: #667eea;
  --primary-rgb: 102, 126, 234;
  --secondary-color: #764ba2;
  --secondary-rgb: 118, 75, 162;
}

/* 绿色主题 */
:root[data-theme="green"] {
  --primary-color: #10b981;
  --primary-rgb: 16, 185, 129;
  --secondary-color: #059669;
  --secondary-rgb: 5, 150, 105;
}

/* 橙色主题 */
:root[data-theme="orange"] {
  --primary-color: #f59e0b;
  --primary-rgb: 245, 158, 11;
  --secondary-color: #d97706;
  --secondary-rgb: 217, 119, 6;
}
```

然后在 HTML 中动态切换主题：

```javascript
// 在 App.jsx 中
document.documentElement.setAttribute("data-theme", "green");
```

## 🐛 常见问题

**Q: 改完颜色后按钮还是老颜色？**  
A: 清除浏览器缓存（Ctrl + Shift + Delete）或强制刷新（Ctrl + Shift + R）

**Q: 某些地方没有变色？**  
A: 可能是有其他 CSS 文件覆盖了，检查 `App.css` 或页面特定样式

**Q: 想改回原来的颜色？**  
A: 按相反顺序再替换一遍即可
