## 设计特点

### 新配色方案的优势

1. **专业感**：深蓝色传达信任和专业
2. **可读性**：提升的对比度改善文本可读性
3. **现代感**：简洁的灰度系统看起来更现代
4. **可访问性**：更好的 WCAG 对比度合规性
5. **一致性**：所有组件使用统一的颜色系统

### 新配色结构

- **主色**：#1e3a8a（深蓝色）- 用于主要交互元素
- **副色**：#1e40af（深蓝色变体）- 用于悬停和高亮
- **背景**：#f3f4f6（淡灰色）- 用于卡片和容器背景
- **边框**：#d1d5db（浅灰色）- 用于分割线和边框
- **文本**：#333（深灰色）- 主文本，#4b5563（中灰色）- 次要文本
