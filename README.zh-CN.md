Easy ICS 是一款轻量级的全栈应用，用于把图片和自然语言文本中的日程信息智能转换为标准的 ICS（iCalendar）文件，省去把截图或纸质日程逐条手输进日历的麻烦。它支持将 PNG/JPG/JPEG/BMP/TIFF 等图片通过 Tesseract OCR 提取文字，再由解析器识别事件标题、日期时间、地点与描述，并生成完全符合 RFC 5545 的 .ics 文件，可在 Google 日历、Outlook、Apple 日历等平台无缝导入；同时也能直接解析自由文本中的日程信息。前端基于 React + Vite，后端采用 FastAPI + Pydantic + Uvicorn，提供简洁的一体化界面（拖拽/粘贴图片、实时进度、结果预览与一键下载）与开发者友好的 REST API（如健康检查、图片上传 OCR、文本解析与 ICS 生成），并配有 Swagger 与 ReDoc 交互式文档。当前版本（v0.1.0）已包含图像识别、文本解析、ICS 生成、Web 界面与 API；后续计划包括更强的文本解析、集成测试、Docker 化与性能优化，以及在 v1.0.0 中加入重复事件、多人日历导出、鉴权、事件历史与移动端应用。项目采用 MIT 许可，开源、跨平台、隐私友好，既适合个人直接使用，也便于集成到现有系统中。
