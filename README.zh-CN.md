# Easy ICS

[English](README.md) | [中文](README.zh-CN.md)

## About

Easy ICS 是一款轻量级应用程序，可将图像和自然语言文本中的日程信息转换为标准 ICS（iCalendar）文件，省去手动逐一将截图或纸质日程输入日历的繁琐操作。支持从 PNG/JPG/JPEG/BMP/TIFF 图像中提取文本，解析事件标题、日期/时间、地点及描述，生成完全符合 RFC 5545 规范的.ics 文件。这些文件可无缝导入 Google 日历、Outlook、Apple 日历等平台。同时支持直接解析自由格式文本中的日程信息。未来规划包括增强文本解析能力、集成测试、Docker 化部署、性能优化，并在 v1.0.0 版本中新增重复事件、多用户日历导出、身份验证、事件历史记录及移动应用功能。本项目采用 MIT 开源许可证，支持跨平台运行且注重隐私保护，既可直接用于个人场景，亦能轻松集成至现有系统。
